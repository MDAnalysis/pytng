List of Available Blocks¶

Platform independent definitions for the block IDs can be found below. You should NOT use these directly. Instead use the TNGFileIterator.block_ids attribute to see the blocks and block IDs available in your TNG file.

GROUP 1: Standard non-trajectory blocks¶

Block IDs of standard non-trajectory blocks.

Block Name

Block ID (long long)

0x0000000000000002LL
